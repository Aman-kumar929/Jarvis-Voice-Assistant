# JARVIS – Python Voice Assistant

A lightweight **Python-based voice assistant** that listens for a wake word (“Jarvis”),
understands voice commands, and performs actions like opening websites and playing music.

---

## About
**JARVIS** is a simple voice-controlled assistant built using Python. It uses speech recognition
to capture user commands and text-to-speech to respond back. After activation, it can open popular
websites and play songs from a custom music library.

This project demonstrates **Python automation**, **speech recognition**, and **voice-based interaction**.

---

## Features
- Wake-word detection (“Jarvis”)
- Speech-to-text using Google Speech Recognition
- Text-to-speech responses using `pyttsx3`
- Voice-controlled web browsing
- Music playback using a custom library

---

## Tech Stack
- Python
- SpeechRecognition
- pyttsx3
- Webbrowser
- PyAudio

---

## Example Commands
- Jarvis open Google
- Jarvis open YouTube
- Jarvis play song-name
